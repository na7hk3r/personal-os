# Atajos de teclado — Personal OS

| Atajo | Acción |
| --- | --- |
| `Ctrl/Cmd + K` | Abrir/cerrar Command Palette (búsqueda global) |
| `↑` / `↓` | Navegar resultados en el palette |
| `Enter` | Abrir resultado seleccionado |
| `Esc` | Cerrar palette o modal activo |

## Próximos atajos planeados (roadmap)

- `Ctrl + N`: Nueva nota
- `Ctrl + T`: Nueva tarea en Kanban
- `Ctrl + F`: Iniciar sesión de foco
- `Ctrl + B`: Toggle sidebar
