# Capturas de pantalla

PNGs reales que usa la landing:

- `screenshot-dashboard.png` - Vista del dashboard principal.
- `screenshot-copiloto.png` - Panel del copiloto IA local.
- `screenshot-work.png` - Plugin Work.
- `screenshot-temas.png` - Tema oscuro.
- `screenshot-temas-light.png` - Tema claro.

La seccion de temas usa `screenshot-temas.png` y `screenshot-temas-light.png`
como una sola comparacion interactiva con linea diagonal.

Tamano recomendado: ratio cercano a 16:10, formato PNG optimizado.

Si una captura falta, la landing renderiza un placeholder discreto en su lugar.
